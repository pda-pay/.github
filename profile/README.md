# 기능소개
## 고객 클라이언트
- 보유 주식 보여주기
- 보유 주식 담보로 설정하기
- 담보 주식 반대매매 우선순위 설정하기
- 고객의 보유주식/담보주식 보여주기
- QR 인식을 통한 결제하기
- 결제, 상환, 반대매매 등 알림 보내기
## 가맹점 클라이언트
- 금액 입력을 통한 QR코드 생성하기
## 어드민 클라이언트
- 상환, 반대매매 등 발생 시 보여주기
- 고객의 담보 비율 보여주기
